# Design Tokens

Every visual decision in NubiscoUI is backed by a CSS custom property. This page renders them directly from the compiled SCSS source, so what you see here matches exactly what ships in the library.

The token data is also published as a machine-readable file at [`/ui/tokens.json`](/ui/tokens.json) in [W3C Design Token (DTCG)](https://tr.designtokens.org/format/) format. Figma plugins such as [Tokens Studio](https://tokens.studio) can sync directly from that URL.
